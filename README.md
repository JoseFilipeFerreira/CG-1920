# OpenGL Engine & Primitive Generator
OpenGL based engine that parses a XML file containing a scene

## Usage
* install with:
```
bash build.sh
```
* run generator with:
```
./target/release/bin/generator --help
```
* run engine with:
```
./target/release/bin/engine input_file
```
you can find examples [here](scenes)
